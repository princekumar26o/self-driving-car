# abstraction

in this file we have written a abstraction code of self driving car in which we mainly focuses on basic function of a car such as startengine, shiftgear,accelerate,apply brake and stop engine of a car. abstraction mainly hides unnecessary details from a client and showcase only what is necessary.

# Encapsulation 

it is one of the pillars of a object oriented programming, in this concept we learn how to wrapped data(variables) and methods(function) into a single class and restict the direct access of data from a client using access modifiers.
